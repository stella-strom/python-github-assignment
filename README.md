# Work Time Tracker

This program allows the user to enter how many hours they worked today and then calculates an estimate of how many hours they will work for a standard 5-day workweek. It's designated to help users track productivity, monitor workload, and get a quick idea of their weekly commitment. The program also includes basic error handling to prevent crashes from invalid input.
python app.py
Enter hours worked today: 4
You are on track to work 20 hours this week.
